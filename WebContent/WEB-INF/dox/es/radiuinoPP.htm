Settings

Every Radiuino data point requires a Name.

The Sensor address indicates the address of the sensor that will provide the data for this data point.

The Radiuino data type indicates what type of data is being received in the Radiuino packet.

The Byte index (Radiuino packet) indicates which is the first byte within the Radiuino packet where the data for this data point is located.

The Multiplier allows to multiply the received value by a factor defined by the user.

The Offset allows to add the value received by a factor defined by the user.

In the Advanced item we have other options that should only be used by those who know the internal structure of Radiuino. Do not use if you are not familiar with the Radiuino structure.

The Byte map in the polling package allows you to define fixed values within the polling packet.

The Byte map in the request package allows you to define fixed values within the value sending package.
