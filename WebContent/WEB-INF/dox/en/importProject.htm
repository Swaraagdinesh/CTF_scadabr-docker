Summary

This functionality allows you to import a complete project (.zip) into ScadaBR. Attention: all data of the current project will be erased!

Importing
Select the project's .zip file on your computer and click the Submit button. ScadaBR will receive the file and show its basic information such as Project Name and Description. Make sure it is the correct project and start the import (which may take a few minutes).
Precautions
* After starting the import, wait until its completion.
* Don't forget that all data in the current project will be erased.
* If the project has been exported with the history of point values, the process can be take several minutes.
* The current users of the system will be deleted and those of the exported project will be inserted. Make sure that you know at least one username and password for the new project so that you can log-in into the system.
