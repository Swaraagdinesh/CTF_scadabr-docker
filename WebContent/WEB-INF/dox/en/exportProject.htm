Summary

Use this functionality to export your current project configured in ScadaBR. A .zip file with all the files needed for later recovery of this project will be created.

Project Name and Description
The name and description of the project are only used to identify the project when it is being imported.
Include point values
The data referring to value of points in ScadaBR database can take up a lot of disk space. Only select this option if it is really necessary to export the value history together with the rest of the project. Exporting (and mainly importing) point values can take a considerable amount of time.
Include Uploads folder
The Uploads folder is the ScadaBR folder that contains the data that you have sent to the system, such as the background pictures of the graphical views, for example.
Include Graphics folder
The Graphics folder is the ScadaBR folder that contains the figures used in the graphical views. Only include this option if you have manually added new graphics components to the system.
