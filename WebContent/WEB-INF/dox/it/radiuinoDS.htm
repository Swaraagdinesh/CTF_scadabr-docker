Description

This datasource is used to read data from a serial connection to a Radiuino base. The system will search for the sensors defined in the data sources and collect the information according to the selected bytes.

Settings

Every data source requires a Name.

The Port parameter must indicate which serial port is being used for connection.

The Baud Rate parameter indicates the data transmission speed.

The Data Bits parameter indicates how many bits are transmitted at a time.

The Stop bits parameter indicate how many stop bits there will be.

The Parity parameter indicates whether or not there will be parity.

Polling mode indicates whether the system should poll to collect information or not polling and waiting for sensors to send data periodically.

The Update period parameter indicates the time interval that the system will poll for new data.

The Retries number parameter indicates how many polling attempts will be carried out.

The Timeout (ms) parameter specifies how long each polling attempt will wait to be successful. (Time in milliseconds)
